# CellLifeSimulation

Project of 2D cell life simulation with C++ and C#

I realised it with MVC pattern

There are five projects here :

- C++ static library for simulation model
- C++/CLR DLL wrapper
- Qt C++ client
- WPF C# client
- Web C# ASP.NET / Angular client
